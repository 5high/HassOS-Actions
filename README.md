# HassOS-Actions

本项目为自动云编译HassOS   
每日凌晨编译最新的stable版本HassOS    
提供Green，Yellow和树莓派系列以及虚拟机镜像下载

# 固件特点

1. 增加HTOP和VIM，NANO命令。
2. 固件支持DoT解析DNS信息
3. 优化GitHub访问
4. 大陆地区可正常启动并安装 （耗时大约1小时左右）
5. 集成HACS插件商店
6. 集成ItCommander加载项安装源
7. 集成Zigbee2mqtt加载项安装源
