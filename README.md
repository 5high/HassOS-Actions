# HassOS-Actions

本项目为自动云编译HassOS   
每日凌晨编译最新的stable版本HassOS    
由于虚拟机镜像过大云编译会报空间不足无法成功编译，所以只提供树莓派各系列的镜像下载。

# 固件特点

1. 增加HTOP和VIM，NANO命令。
2. 更改固件系统时区为+8区（中国大陆地区）。
